# Privacy

Steady Voice AI Studio is local-first.

## Defaults

- No telemetry by default
- No upload without user action
- Browser localStorage for local MVP persistence
- Manual API key handling only

## Placeholder controls

- secret vault
- environment template
- backup export
- data deletion

## Data handling goals

- Keep the app usable without a network connection.
- Only store what the user chooses to save.
- Make deletion obvious and user-controlled.

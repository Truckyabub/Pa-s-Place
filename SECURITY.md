# Security Policy

## Project Scope

Steady Voice AI Studio is a local-first music creation workspace intended for private development and future licensed deployment.

## Security and Privacy Commitments

- No backend services
- No database
- No authentication system
- No tracking scripts
- No cookies are set by this site
- No analytics tools are included
- No payment processor is included
- No live forms are included
- No unsafe third-party embeds are used

## Future Changes

If contact links or other data collection features are added later, they should be reviewed carefully before launch and should collect only the minimum information needed.

## Reporting Concerns

If a security or privacy issue is discovered, keep the site private until the concern is reviewed and resolved.

# Security Policy

## Site Status

Steady Voice AI Studio is a local-first application. It should remain private where needed and free of unsafe backend or upload behavior by default.

## Current Security Boundaries

- No backend.
- No database.
- No financial transactions.
- No API keys.
- No secrets.
- No login system.
- No public admin panel.
- No public analytics.
- No live forms.
- No public QR codes.

## Personal Information

Do not publish private names, addresses, phone numbers, emails, or any private data that has not been approved for public viewing.

Private addresses must never be displayed publicly.

## Before Enabling Any Form

Before any form becomes active:

- Confirm who receives submissions.
- Confirm storage and deletion rules.
- Confirm access rules.
- Confirm privacy language.
- Test the form privately.
- Log approval from the approved family steward.

## Payments Later

Do not collect credit card or banking details directly in this app.

## Secrets

Never commit `.env` files, API keys, private tokens, private config files, payment credentials, or personal contact lists.

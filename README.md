# Pa's Place

Pa's Place is a static family tribute website honouring George Alexander Carey, lovingly known as Pa George.

## What is included

- `index.html` - main website
- `styles.css` - screen styling
- `script.js` - navigation, gallery, copy, print, and accessibility behavior
- `assets/css/print.css` - print-only event styling
- `assets/images/` - placeholder image folder
- `CONTENT_GUIDE.md` - family-safe update prompts
- `PRIVACY_AND_SAFETY.md` - publishing safety checklist
- `.nojekyll` - GitHub Pages compatibility marker

## Local use

Open `index.html` in a browser.
No build step is required.

## Update guidance

Only publish family-approved content.
Do not add private contact details, payment links, fake sponsors, copyrighted images, or unverified charity claims.

## GitHub Pages

Deploy from the `main` branch and the repository root.
